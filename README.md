PopX Authentication App

A responsive React application with Firebase Authentication for Signup, Login, and Protected Profile routes. Designed for clean UI, form validation, and secure user authentication.

## Features

- *Signup & Login* using Firebase Auth (Email & Password)
- *Protected Profile Page* – accessible only after login
- *Form Validations* – real-time feedback on empty or invalid fields
- *Responsive UI* – mobile-first design, works perfectly on any device
- *Firebase Integration* – secure authentication without building backend from scratch
- *Routing* – React Router with protected routes

## Tech Stack

- *Frontend:* React, CSS
- *Routing:* React Router v6
- *Authentication:* Firebase Auth
- *Version Control:* Git & GitHub
